Required Files:
QuestionMain.java
QuestionTree.java
QuestionNode.java
UserInterface.java

QuestionMain.java
  This file serves as the main executable code for the 20 Questions game.
  
QuestionTree.java
  This file creates a binary tree resembling the questions & answers for a game of 20 questions.
  
QuestionNode.java
  This file creates individual nodes within the binary tree represented by QuestionTree.
  
UserInterface.java
  This file provides framework for the above QuestionMain file.
